# AIdiver 記事サムネイル制作スキル

このスキルを呼び出したら、以下のステップでサムネイル仕様書を作成する。

---

## 参照デザインパターン

以下の2パターンがAIdiverメディアの標準スタイル。制作前に記事のトーンに合わせて選択する。

### パターンA：ダーク・ドラマ型
参考: `https://ez-cdn.shoeisha.jp/static/images/article/23897/23897-ogp.png`

| 要素 | 仕様 |
|------|------|
| 背景 | 黒〜チャコールグレーのグラデーション |
| ビジュアル | 人物シルエット・AIロボット・モノクロ写真を右寄せ配置 |
| メインタイトル | 大きなボールド日本語ゴシック。白または黄色 |
| アクセント | 黄色・オレンジ系（強調ワードや鉤括弧部分に使用） |
| テキスト配置 | 左寄せ縦並び。連載ラベルを右上に小さく配置 |
| トーン | 緊張感・問題提起・セキュリティ・リスク系の記事に合う |

### パターンB：テックブルー型
参考: `https://cz-cdn.shoeisha.jp/static/images/article/23340/AI1200_ogp.png`

| 要素 | 仕様 |
|------|------|
| 背景 | ディープネイビー〜ブルーのグラデーション |
| ビジュアル | 光るAIロボット・ホログラム・六角形パターンを中央〜右配置 |
| メインタイトル | 大きなボールド白ゴシック。3行以内 |
| アクセント | ブルーグロウ・シアン系の発光エフェクト |
| テキスト配置 | 左寄せ縦並び。ブランドロゴを右下または左上に |
| トーン | AI開発・テクノロジー・先進性・ポジティブ系の記事に合う |

---

## Step 1: ヒアリング

以下を確認してから仕様書を出力する。

| 項目 | 確認内容 |
|------|---------|
| 記事タイトル | サムネイルに載せるメインテキスト |
| サブコピー | 補足テキスト（任意、1行以内） |
| 連載名 | 連載ラベルの有無と名称（任意） |
| デザインパターン | A（ダーク）/ B（テックブルー）/ 未定 |
| 使用画像素材 | 用意済みの写真・イラスト（あれば） |
| 出力形式 | SNS OGP（1200×630px）/ YouTube（1280×720px）/ 正方形（1080×1080px） |
| 制作ツール | Canva / Photoshop / Figma / 外部発注 |

## Step 3: Canva制作時の推奨手順

Canvaで制作する場合の手順を案内する。

1. **新規デザイン作成**
   - `カスタムサイズ` から指定サイズ（例: 1200×630px）を入力
2. **背景設定**
   - パターンA: 背景色 `#1a1a1a` → グラデーション追加（チャコール系）
   - パターンB: 背景色 `#0a1628` → グラデーション追加（ネイビー→ブルー）
3. **素材配置**
   - Canva素材検索: 「AI robot」「cybersecurity」「hologram」等でキーワード検索
   - 右寄せに配置し、左端をフェード処理（透明度グラデーション）
4. **テキスト入力**
   - フォント: Noto Sans JP Bold または BIZ UDPGothic Bold
   - 仕様書のサイズ・色に従って配置
5. **最終確認**
   - 「プレビュー」でSNS表示イメージを確認
   - PNG（高品質）でエクスポート

---

## Step 4: 完成チェック＆ハンドオフ

仕様書出力後・制作完了後に以下を確認する。

- [ ] テキストが3行以内に収まっているか
- [ ] 端から40px以上のマージンが確保されているか
- [ ] コントラスト比が十分か（暗背景＋白文字が基本）
- [ ] AIdiverロゴが正位置に入っているか
- [ ] PNG形式・指定サイズで書き出されているか
- [ ] writerに仕様内容と記事タイトルの整合を確認したか

完成データは `data/documents/` に保存し、writerとeditorに共有する。
